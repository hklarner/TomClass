7.2. Core/CPLanguage.py¶

class CPLanguage.Converter(Messenger, Model)¶
Parameters:
  • Messenger – A Session.Messenger object.
  • Model – A Session.Model object.

The following method computes all parameter indeces that are referenced by a state formula and a component.

parameter_reference(StateFormula, Component)¶
Returns:...
Return type:...

The following 8 methods convert predicates expressions to low-level constraints. Most of them require the above method parameter_reference() to do so. The parameters and parameter types that the predicate methods share, are listed below.

Predicate parameters

Parameters:
  • Component (String) – The component used for parameter reference.
  • StateFormula (String) – A state formula.
  • Operator (String) – One of '<','<=','=','>=','>','!='.
  • Activity (Integer) – An activity.
  • Quantifier (String) – One of 'All','Some'
  • Label (String) – One of 'Activating','ActivatingOnly','Inhibiting','InhibitingOnly','Observable'
  • Threshold – The interaction threshold of the label.
  • Graph (Dictionary) – A successor-based representation of a subgraph of the unitary and asynchronous state transition graph. Each key of the dictionary is a state. The value of a key is a list of successor states.
  • Path (List of states) – A list of states, representing a path in the unitary and asynchronous state transition graph.
inequality_abs(Quantifier, StateFormula, Component, Operator, Activity)¶

For semantics see Predicate Type 1: Inequalities.

inequality_rel(Quantifier, StateFormula1, Component1, Operator, StateFormula2, Component2)¶

For semantics see Predicate Type 1: Inequalities.

identity(Quantifier, StateFormula, Component, Operator, Activity)¶

For semantics see Predicate Type 2: Identities.

multiplex(StateFormulas, Component)¶

The “compounds” of a multiplex. The component that is regulated by the multiplex. For semantics see Predicate Type 3: Multiplexes.

edge_label(Label, Regulator, Target, Thresholds, StateFormula="True")¶

The optional state formula that restricts the contexts must satisfy the constraint (see edge_label predicate). The default True enforces no restriction. The regulator component of the interaction. The target component of the interaction. For semantics see Predicate Type 4: Edge Labels.

For semantics see Predicate Type 5: Subgraphs.

For semantics see Predicate Type 5: Subgraphs.

For semantics see Predicate Type 6: Comparisons.
